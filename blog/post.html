Added code on bitbucket and will be working off of there.

Alex on October 25, 2015

Worked with a friend named John Reinstra who has been teaching me how to properly implement and work with a lot of this code. I'm working off of the backend of Dragonfly which John Reinstra set up.

The algorithm starts by breaking apart url string into different parts to analyze them. This is kind of a hacky way to get data. The specific code is as follows:
$q = urldecode($_GET["q"]);
$uncut = explode(" ", $q);

Next the algorithm goes through each word exploded from the array and makes sure it actually has content. It does this through iteracting through each one and checking for a length greater than 0. foreach($uncut as $word) { if(strlen($word) > 0) { $words[] = $word; } }

After the algorithm loops through each word in the array, with foreach($words as $word), it preps the SQL queries with variations of mysqli_prepare($con, $sql) . After sending the quaries it fetches the results with mysqli_stmt_fetch
